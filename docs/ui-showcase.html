2. Settings — Saved keys are never re-displayed in full

Reopening the settings page after a key has been saved leaves every key field blank with a masked "ending in" hint instead of repopulating the literal secret. Leaving a field untouched and changing something else (e.g. provider, results count) no longer overwrites the saved key with an empty value — only typing a replacement does. There is no way to view a saved key in full again through the extension.
