Core loop

Open the panel. Get back to reading.

01 — READ

Reads the page you're on

A one-shot script reads the readable text of the active tab only — never tabs you haven't clicked on, never running in the background.

02 — SEARCH

Searches with your provider

That text goes to your chosen AI provider, which uses its own web search to find related pages. No separate search API key needed.

03 — RANK

Ranks and explains

Each result comes back with a short, honest note on why it's relevant — the product's actual trust signal, not a persuasive flourish.

04 — RESULT

Click through, or don't

Past sessions persist in History. Reopening one re-renders instantly — no new searches, no new cost.

No backend. No accounts. No telemetry.

Runs entirely in your browser. There's no server the developer operates, and no account to create.

Your keys, your connection. Page content and credentials go straight from your browser to the provider you configured — Anthropic, OpenAI, Gemini, or OpenRouter.

Local storage only. Settings, API keys, and your last 50 research sessions live in chrome.storage.local. Nothing syncs.

Active tab only. It reads the page you click the toolbar icon on — nothing else, and nothing in the background.

Works with the provider you already trust.

Anthropic OpenAI Gemini OpenRouter

Paste an API key directly, or click Connect to OpenRouter to sign in without pasting one. Each provider uses its own web search to find related pages, so result quality can vary slightly depending on which one you pick.
